# Nomadi Circle

**Nomadi Circle** is a next-generation social travel platform designed to connect travelers, help them plan trips, and discover new experiences. Built with Next.js 16, TypeScript, and Tailwind CSS.

## 🚀 Features

### 1. Social Feed (`/feed`)
- **Stories**: Share ephemeral travel moments.
- **Posts**: Share photos and trip summaries.
- **Interactions**: Like, comment, and save posts.

### 2. Destination Discovery (`/explore`)
- **AI Planner**: Generate itineraries based on your vibe.
- **Destination Wiki**: Detailed guides for places like Manali, Goa, and Spiti.
- **Filters**: Find places by region, budget, and vibe.

### 3. Community (`/community`)
- **Group Chats**: Connect with travelers in real-time.
- **Private Groups**: Plan trips with friends.

### 4. Travel Partners (`/go-out`)
- **Partner Finder**: Find people to travel with.
- **Post Needs**: Request travel buddies for specific trips.

### 5. Festivals (`/festivals`)
- **Calendar**: Discover cultural events by month.
- **Event Details**: Get the vibe check and pro tips.

### 6. Stays (`/stays`)
- **Accommodation Finder**: Hostels, homestays, and hotels.
- **Booking Flow**: Seamless booking experience.

### 7. Global AI Assistant
- **Nomadi AI**: A floating assistant to help you navigate the app and plan trips.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Components**: Custom Design System (Radix UI primitives)

## 🏃‍♂️ Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open the app**:
    Visit [http://localhost:3000](http://localhost:3000)

## 🏗️ Project Structure

- `src/app`: Next.js App Router pages and API routes.
- `src/components`: Reusable UI components.
- `src/features`: Feature-specific components and logic.
- `src/lib`: Utilities and helpers.
